# Pokemon Async / Await Assignment

This project uses async/await and the Pokemon API to fetch data for a random Pokemon from the original 151 Pokemon.

## What it does

- Generates a random Pokemon ID from 1 to 151
- Fetches Pokemon data from the Pokemon API
- Makes a second API call for species data
- Returns an object with:
  - name
  - height
  - weight
  - types
  - flavorText
  - habitat
  - isLegendary

## How to run

```bash
node index.js
```
